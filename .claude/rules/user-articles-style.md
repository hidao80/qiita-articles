# ユーザーが作成する記事のスタイルルール

- `articles/*.md`内の文章の文体に似せてください。言葉遣い、単語のチョイス、雰囲気、句読点の位置、改行の基準を強く意識すること。
- 文体は既存の記事にかかわらず「です・ます調」に統一すること。ただし、必要に応じて漢字をカタカナに開いたり、体言止めにすることができます。
- 必要がない限り、記事の見出しおよび本文中にアルファベットややラテン文字を使わないこと。
- 参考文献や元情報に敬意を払う表現を取ること。
- テンプレートファイルを指定されたときは、テンプレートに従い、入力された情報を整理してプレースホルダーを埋めるように記述すること。
- 参考にする情報は1次情報を原則とします。ただし、Zenn記事の場合は他のユーザーのZenn記事を、Qiita記事の場合はQiita記事を1次情報に次ぐ情報源として使用することができる。
- スクリーンショットやサンプルコードが多い、または長い場合はHTMLの`<summary>`タグをを使ってアコーディオンにしてください。

場当たり的に解決できたのでメモ。
詳細が分かる方は編集リクエストかコメントください。🙇

## TL;DR
```sh:pnpm install
pnpm install --shamefully-hoist
```

## 環境
**OS**: Windows 11 Home 23H2
**Laravel**: 8.83.27
**Node.js**: v12.19.0
**pnpm**: 5.18.10

## 雑な説明

パッケージのインストール時、シンボリックリンクに対応するオプションを指定すると`node_modules/`以下が`npm install`のファイルパスとほぼ同じになる。

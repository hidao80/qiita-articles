新しく環境をセットアップするときには、全体をコピペして使います。

```bash
brew tap heroku/brew
brew tap adoptopenjdk/openjdk
brew tap gcenx/wine
brew install heroku git 7zip composer wget pandoc ffmpeg node mkcert 
brew install --cask google-chrome visual-studio-code vlc libreoffice brave-browser \
      discord gimp　inkscape dbeaver-community vnc-viewer virtualbox vagrant \ 
      veracrypt xmind zoom microsoft-edge microsoft-teams google-japanese-ime \
      amorphousdiskmark  obs barrier adoptopenjdk8 diffmerge skitch wine-crossover \
      background-music
```

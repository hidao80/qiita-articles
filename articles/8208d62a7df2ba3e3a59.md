新しく環境をセットアップするときには、全体をコピペして使います。

```powershell
# 通信
cinst -y googlechrome brave discord ultravnc zoom advanced-ip-scanner microsoft-teams `
         wget bonjour 

# 開発
cinst -y git visualstudiocode composer dbeaver winmerge bzeditor winscp `
         virtualbox vagrant jre8 wsl2 microsoft-build-tools python python3 nodejs `
         mkcert openssl php 

# 画像・動画・音声
cinst -y camtasia Screenpresso vlc gimp InkScape eartrumpet ffmpeg obs 

# 資料作成
cinst -y libreoffice xmind pandoc

# ユーティリティ
cinst -y 7zip veracrypt win32diskimager scansnapmanager googlejapaneseinput cpu-z `
         crystaldiskmark  barrier
```

注記：`vagrant`は`virtualbox`のあとに来るようにしないと正しくインストールされない場合があります。

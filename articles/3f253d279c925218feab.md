コマンドを実行すると`~/.vscode/extensions/`フォルダを掘ってそこに機能拡張を突っ込まれます。
`code`は VSCode のプログラムファイル名です（`code.exe`など）。

下記のコード領域をPowerShellやターミナルなどにコピペすれば、以下のすべての拡張機能を一発でインストールできます！

```sh
# PHP
code --install-extension valeryanm.vscode-phpsab              # php の linter と自動フォーマッター
code --install-extension junstyle.php-cs-fixer                # php の linter と自動フォーマッター（サブ）
code --install-extension felixfbecker.php-intellisense        # php の自動補完
code --install-extension neilbrayfield.php-docblocker         # phpdoc コメントの入力補完

# フロントエンド自動整形
# code --install-extension esbenp.prettier-vscode

# CSS
code --install-extension HookyQR.minify   # JS/CSS の圧縮ツール

# javascript
code --install-extension dbaeumer.vscode-eslint   # JS 用の Linter
code --install-extension lllllllqw.jsdoc  # JS 用の関数コメントヘルパ

# 変換
code --install-extension ahebrank.yaml2json

# VSCode
code --install-extension ms-ceintl.vscode-language-pack-ja  # VSCode 日本語パック

# Markdown
code --install-extension shd101wyy.markdown-preview-enhanced  # Markdown の高性能プレビュアー
code --install-extension tht13.html-preview-vscode            # Markdown を HTML で出力する
code --install-extension yzane.markdown-pdf                   # Markdown を PDF で出力する
code --install-extension marp-team.marp-vscode                # Markdown でスライドを作成し、PDF で出力する

# editconfig
code --install-extension editorconfig.editorconfig  # テキストファイル全般の自動フォーマッター

# excel
code --install-extension GrapeCity.gc-excelviewer  # Excel ファイルビュアー

# sftp
code --install-extension Natizyskunk.sftp  # sftp 自動/手動アップロードクライアント

## ドキュメント管理
code --install-extension ics.japanese-proofreading     # textlint（日本語校正）
code --install-extension vanadium23.markdown-hashtags  # ハッシュタグ検索
code --install-extension oderwat.indent-rainbow        # インデントハイライト

# コーディング補助
#code --install-extension GitHub.copilot
code --install-extension kiteco.kite
code --install-extension oderwat.indent-rainbow  # インデントの深さを色で判別できるようにする

# Git
code --install-extension eamodio.gitlens   # カーソル位置の行が誰によっていつコミットされたかを表示する

# 作図
code --install-extension hediet.vscode-drawio  # ドローツールの draw.io で作図できる
```

現在インストール済みの機能拡張一覧を表示するコマンド。

```sh
code --list-extensions
```

## さいごに

おすすめの拡張機能がある場合、拡張機能名だけだと重複しているケースが散見されるので、パッケージ名（`ms-ceintl.vscode-language-pack-ja`みたいなやつ）で指定しましょう！

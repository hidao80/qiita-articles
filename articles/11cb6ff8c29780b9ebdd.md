## tl;dr

自分が管理しているサイトにSNSのシェアボタンを設置したいときに利用できる、各SNSのシェア機能用URLをまとめました。

:::note warn
以下の内容は2026年2月22日現在の一部の環境の動作に基づいています。
各サイトの仕様変更によって本記事に記載されている方法では利用できなくなる場合があります。
:::

## 投稿内容例

```text:描画されるテキスト例
#qiita に投稿しました！

AndroidスマホでDockerフリーなコンテナ環境を作る - Podman + podman-compose 実機検証記録 #container - Qiita
https://qiita.com/hidao/items/1c7a9b26757500ff5d9e
```

文章とURLはURLエンコードしてクエリパラメーターにします。
URLエンコードしたりデコードするときは、ブラウザの開発者ツールにあるコンソールでJavaScriptの関数`encodeURIComponent()`や`decodeURIComponent()`を使うと簡単に作れます[^1]。

```sh:URLエンコード例

```

## 各種SNSのシェア投稿URLの構造

### X(旧Twitter)
URLは別段落になります。
```
https://x.com/intent/post?text=${TEXT}&url=${URL}
```
```:例
https://x.com/intent/post?text=%23qiita%20%E3%81%AB%E6%8A%95%E7%A8%BF%E3%81%97%E3%81%BE%E3%81%97%E3%81%9F%EF%BC%81%0A%0AAndroid%E3%82%B9%E3%83%9E%E3%83%9B%E3%81%A7Docker%E3%83%95%E3%83%AA%E3%83%BC%E3%81%AA%E3%82%B3%E3%83%B3%E3%83%86%E3%83%8A%E7%92%B0%E5%A2%83%E3%82%92%E4%BD%9C%E3%82%8B%20-%20Podman%20+%20podman-compose%20%E5%AE%9F%E6%A9%9F%E6%A4%9C%E8%A8%BC%E8%A8%98%E9%8C%B2&url=https%3A%2F%2Fqiita.com%2Fhidao%2Fitems%2F1c7a9b26757500ff5d9e
```

### Threads
アカウントがないため動作確認できていません。
```url
https://www.threads.net/intent/post?text=${TEXT}&url=${URL}
```
```:例
https://www.threads.net/intent/post?text=%23qiita%20%E3%81%AB%E6%8A%95%E7%A8%BF%E3%81%97%E3%81%BE%E3%81%97%E3%81%9F%EF%BC%81%0A%0AAndroid%E3%82%B9%E3%83%9E%E3%83%9B%E3%81%A7Docker%E3%83%95%E3%83%AA%E3%83%BC%E3%81%AA%E3%82%B3%E3%83%B3%E3%83%86%E3%83%8A%E7%92%B0%E5%A2%83%E3%82%92%E4%BD%9C%E3%82%8B%20-%20Podman%20+%20podman-compose%20%E5%AE%9F%E6%A9%9F%E6%A4%9C%E8%A8%BC%E8%A8%98%E9%8C%B2&url=https%3A%2F%2Fqiita.com%2Fhidao%2Fitems%2F1c7a9b26757500ff5d9e
```

### Bluesky
urlパラメーターを指定しても、投稿に反映されませんでした。URLは`text`パラメーターに含めてください。
```url
https://bsky.app/intent/post?text=${TEXT}
```
```:例
https://bsky.app/intent/post?text=%23qiita%20%E3%81%AB%E6%8A%95%E7%A8%BF%E3%81%97%E3%81%BE%E3%81%97%E3%81%9F%EF%BC%81%0A%0AAndroid%E3%82%B9%E3%83%9E%E3%83%9B%E3%81%A7Docker%E3%83%95%E3%83%AA%E3%83%BC%E3%81%AA%E3%82%B3%E3%83%B3%E3%83%86%E3%83%8A%E7%92%B0%E5%A2%83%E3%82%92%E4%BD%9C%E3%82%8B%20-%20Podman%20+%20podman-compose%20%E5%AE%9F%E6%A9%9F%E6%A4%9C%E8%A8%BC%E8%A8%98%E9%8C%B2%0Ahttps%3A%2F%2Fqiita.com%2Fhidao%2Fitems%2F1c7a9b26757500ff5d9e
```

### Mastodon
textの内容の後に改行文字`\n`（`%0A`）があると見栄えが良いです。お好みで。
なお、例示しているURLのドメイン`your.server`はシェアしたいSNSのドメインに置き換えて利用してください。
```url
https://your.server/share?text=${TEXT}%0A&url=${URL}
```
```:例
https://your.server/share?text=%23qiita%20%E3%81%AB%E6%8A%95%E7%A8%BF%E3%81%97%E3%81%BE%E3%81%97%E3%81%9F%EF%BC%81%0A%0AAndroid%E3%82%B9%E3%83%9E%E3%83%9B%E3%81%A7Docker%E3%83%95%E3%83%AA%E3%83%BC%E3%81%AA%E3%82%B3%E3%83%B3%E3%83%86%E3%83%8A%E7%92%B0%E5%A2%83%E3%82%92%E4%BD%9C%E3%82%8B%20-%20Podman%20+%20podman-compose%20%E5%AE%9F%E6%A9%9F%E6%A4%9C%E8%A8%BC%E8%A8%98%E9%8C%B2%0A&url=https%3A%2F%2Fqiita.com%2Fhidao%2Fitems%2F1c7a9b26757500ff5d9e
```

### Misskey
URLは別段落になります。
なお、例示しているURLのドメイン`your.server`はシェアしたいSNSのドメインに置き換えて利用してください。
```url
https://your.server/share?text=${TEXT}&url=${URL}
```
```:例
https://your.server/share?text=%23qiita%20%E3%81%AB%E6%8A%95%E7%A8%BF%E3%81%97%E3%81%BE%E3%81%97%E3%81%9F%EF%BC%81%0A%0AAndroid%E3%82%B9%E3%83%9E%E3%83%9B%E3%81%A7Docker%E3%83%95%E3%83%AA%E3%83%BC%E3%81%AA%E3%82%B3%E3%83%B3%E3%83%86%E3%83%8A%E7%92%B0%E5%A2%83%E3%82%92%E4%BD%9C%E3%82%8B%20-%20Podman%20+%20podman-compose%20%E5%AE%9F%E6%A9%9F%E6%A4%9C%E8%A8%BC%E8%A8%98%E9%8C%B2&url=https%3A%2F%2Fqiita.com%2Fhidao%2Fitems%2F1c7a9b26757500ff5d9e
```

## トラブルシューティング

**ハッシュタグを含むアドレスが動作しない**
`encodeURI()`でエンコードした文字列はハッシュタグの`#`がエンコードされずHTMLのIDのようなURLとしてエンコードされてしまうので、ハッシュタグを含む場合は`encodeURIComponent()`を使いましょう[^1]。

[^1]: Qiita "encodeURI() と encodeURIComponent() の共通点と相違点", Yasushi-Mo, 2024/02/06, https://qiita.com/Yasushi-Mo/items/2ab0a0f335f8524b08d6
